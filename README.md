# rk002
Code for the RetroKits [RK-002 Smart MIDI Cable](https://retrokits.com/shop/rk002/) to map the controls on the Novation Minilogue to control Synth 1 & 2 sound design parameters on Circuit Tracks.

Based on Korg **Minilogue MIDI Implementation** ([source](https://www.korg.com/us/support/download/manual/0/544/2890/))
and Novation **Circuit Tracks Programmer's Reference Guide V3** ([source](https://downloads.novationmusic.com/novation/circuit/circuit-tracks)).
